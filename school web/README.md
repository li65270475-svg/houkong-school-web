# 濠江中学奖项申报系统

> 基于 HTML + CSS + JavaScript + localStorage 的奖项申报管理平台

## 项目简介

本系统旨在解决学校获奖信息人工登记、纸质存档、统计繁琐、查询困难等问题，实现获奖管理全流程数字化。

## 功能特性

### 三大角色
- **学生**：在线填报获奖信息、上传证书、查询个人获奖记录
- **教师**：审核班级学生获奖、批量查询、导出本班数据
- **管理员**：管理奖项分类、审核全部信息、生成统计报表、数据备份

### 五大核心功能
1. **信息录入**：表单填写、上传证书图片
2. **信息审核**：学生提交 → 教师初审 → 管理员终审
3. **信息查询**：按姓名、班级、奖项、时间查询
4. **数据统计**：按年度、班级、奖项类型统计，显示图表
5. **数据导出**：导出 Excel / CSV 报表

## 使用说明

### 学生功能

1. **登录**：使用学号和密码登录
2. **查看获奖**：在"我的获奖"页面查看所有获奖记录
3. **申请录入**：填写获奖信息表单，提交审核
4. **查看详情**：点击记录查看详细信息和审核进度

### 教师功能

1. **登录**：使用工号和密码登录
2. **待审核**：查看本班学生提交的获奖申请
3. **审核操作**：通过或驳回学生的获奖申请
4. **查看记录**：查看本班所有获奖记录
5. **导出数据**：导出本班获奖记录为 Excel 或 CSV

### 管理员功能

1. **登录**：使用 admin / 123456 登录
2. **数据概览**：查看全校获奖统计概况
3. **终审管理**：对教师初审通过的申请进行最终审核
4. **奖项分类**：添加、编辑、删除奖项分类
5. **统计报表**：多维度统计分析获奖情况
6. **数据导出**：按条件筛选并导出数据
7. **数据管理**：备份、恢复、初始化数据

## 技术说明

### 技术栈
- HTML5 + CSS3
- JavaScript (ES6+)
- localStorage 数据持久化
- Font Awesome 图标库

### 数据存储
所有数据存储在浏览器 localStorage 中：
- `haojiang_awards`：获奖记录
- `haojiang_categories`：奖项分类
- `haojiang_classes`：班级信息
- `haojiang_students`：学生信息
- `haojiang_teachers`：教师信息

### 兼容性
- Chrome / Firefox / Edge / Safari 等现代浏览器
- 建议分辨率：1024px 及以上

## 注意事项

1. **数据备份**：重要数据请定期使用管理员的"导出全部数据"功能备份
2. **清除缓存**：清除浏览器数据会导致 localStorage 数据丢失
3. **多用户**：同一浏览器可切换不同账号登录（建议使用无痕模式）
4. **证书图片**：图片以 Base64 格式存储，占用存储空间较大
